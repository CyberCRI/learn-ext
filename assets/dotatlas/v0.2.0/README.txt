
Carrot Search dotAtlas
----------------------

Carrot Search dotAtlas is a WebGL-based component for interactive
exploration of large numbers of 2d points.


Contents of this package
------------------------

./carrotsearch.dotatlas.js
  dotAtlas implementation, required at runtime.

./carrotsearch.dotatlas.effects.js
  Experimental utility for animated rollout and pullback of new data sets.

./cookbook/
  Code examples.


Code examples
-------------

Start built-in HTTP server (requires Java):

bin/nhttpd.sh -p 8080 (Linux, Max)

bin\nhttpd.bat -p 8080 (Windows)

Open http://localhost:8080/cookbook/ in your browser and follow the links for code examples.


Documentation
-------------

No documentation yet. By the time dotAtlas gets proper documentation, please use cookbook-style
